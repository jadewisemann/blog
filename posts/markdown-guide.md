# Markdown 작성 가이드

이 가이드에서는 블로그 포스트를 작성할 때 사용할 수 있는 Markdown 문법을 소개합니다.

## 텍스트 스타일

- **굵게** - `**텍스트**`
- *기울임* - `*텍스트*`
- ~~취소선~~ - `~~텍스트~~`
- `인라인 코드` - `` `코드` ``

## 링크와 이미지

[링크 텍스트](https://example.com)

```markdown
[링크 텍스트](URL)
![이미지 설명](이미지 URL)
```

## 리스트

### 순서 없는 리스트
- 항목 1
- 항목 2
  - 중첩 항목

### 순서 있는 리스트
1. 첫 번째
2. 두 번째
3. 세 번째

## 인용문

> 인용문은 `>` 기호로 시작합니다.
> 여러 줄도 가능합니다.
